# Features

这份文档只作为功能文档入口，不再直接维护具体状态内容。

## 功能文档拆分

- 已完成功能：见 `docs/system-state.md`。
- 正在做：先看 `docs/in-progress.md` 了解当前有哪些事项正在推进，再按入口进入 `docs/plan/` 下的对应步骤计划。
- 未来计划：见 `docs/roadmap.md`。

## 维护规则

- 新功能进入开发后，先记录到 `docs/in-progress.md`。
- 需要拆分执行步骤时，在 `docs/plan/` 下新增对应计划文件，并从 `docs/in-progress.md` 链接过去。
- 功能稳定完成后，将记录迁移到 `docs/system-state.md`。
- 暂未开始但已经明确的后续方向，记录到 `docs/roadmap.md`。
- 本文件只维护拆分说明，不再展开功能细节。
