# Roadmap

这份文档用于维护已经明确、但暂未进入当前开发周期的后续计划。

## 未来计划

- 完整生产级认证体系：补齐 token 持久化、过期刷新、主动撤销和跨实例共享能力。
- 更统一的音频协议能力：评估是否为 `tts_audio_chunk` 增加显式音频格式元数据或协商机制。
- 电话与 SIP 扩展：为后续电话网关、SIP trunk、DTMF、IVR 和转人工保留演进空间，但不作为当前核心目标。
- 外部业务平台集成：后续如有需要，再评估知识库、CRM、运营后台和外部 Agent 编排能力的集成边界。
- 更完整的生产级压测体系：补齐高并发、长连接、持续打断和弱网场景下的系统稳定性验证。
